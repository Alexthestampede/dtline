## Critical Conventions

### Scale Factor System (Causes Server Crashes If Wrong)
- `ImageGenerationConfig(width=512, height=512)` takes **pixels**
- The client converts to scale units internally: `scale = pixels // 64`
- For SDXL models: `scale = pixels // 128`
- Direct FlatBuffer users must do this conversion manually
- Server crash symptom: `[cnnp_reshape_build] dim: (2, 262144, 320)` (should be 4096)

### Generated Code — Do Not Edit
- `imageService_pb2.py` / `imageService_pb2_grpc.py` — generated by `grpcio-tools` from `imageService.proto`
- `GenerationConfiguration.py` — auto-generated FlatBuffer schema (see comment at line 73)
- Edits to proto schema go to [drawthingsai/draw-things-community](https://github.com/drawthingsai/draw-things-community)

### Edit/Kontext Models — Required Settings
- `strength=1.0` is **mandatory** (setting lower corrupts the reference encoding)
- Input image must match `width` × `height` exactly
- Must encode as uncompressed float16 (`compress=False` in `tensor_encoder`)
- `_encode_image()` does this automatically; direct callers must pass `compress=False`

### Tensor Format (CCV NNC)
```
[68-byte header][pixel data]
Header: 17 x uint32 little-endian
  [0] magic: 0 = uncompressed float16, 1012247 = fpzip float32
  [1] memory_type: 0x1 = CPU
  [2] format: 0x2 = NHWC
  [3] datatype: 0x20000 = CCV_16F, 0x10000 = CCV_32F
  [5] batch=1, [6] height, [7] width, [8] channels=3
Pixel formula: float_value = uint8 / 127.5 - 1.0  (range -1.0 to +1.0)
```

### TLS Certificate — Required for Secure Connections
**TLS will fail without the Draw Things root CA certificate.** Download `root_ca.crt` from:
https://github.com/drawthingsai/draw-things-community/tree/main/Libraries/BinaryResources/Resources

Place it as `root_ca.crt` in the working directory, or pass the path via `ssl_cert_path`.

### gRPC Channel Options
The client sets large message lengths (32 MB) and HTTP/2 keepalive params. TLS is the default; use `insecure=True` for plain TCP. Self-signed certs: pass `ssl_cert_path` or place `server_cert.pem` in the working directory.

## Important Constraints from README

1. **Kontext/edit models**: `strength=1.0`, input dims = output dims, uncompressed float16
2. **FLUX guidance**: Use `guidance_embed` (default 3.5), not `cfg_scale` (set to 1.0)
3. **TeaCache**: ~20-30% speedup for FLUX at `tea_cache_threshold=0.06`
4. **Tiled diffusion**: Required for images > 1024×1024 to avoid OOM
5. **fpzip required**: `pip install fpzip` or decoded images will raise an error
